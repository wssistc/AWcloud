定时快照

数据被写入了云硬盘之后，可以定期创建快照作为数据备份使用。定时快照使用增量的方式创建快照，即仅创建与上一次快照相比更改了的数据，因此在数据量改动不大的情况下能够在较快的时间内完成快照的创建。尽管快照是以增量方式保存的，但是快照删除流程不会互相制约：未删除的快照均能将云硬盘恢复至该快照状态。

常见的策略如下：

  • 非核心业务数据盘，可设定每周一零点进行一次自动快照备份，保存1个月后自动删除。
  • 核心业务数据盘，每隔12小时进行一次快照备份，保存1周后删除。

修改定时快照任务

  1. 在计划任务列表页，勾选要编辑的定时快照任务，单击“编辑任务”按钮。
  2. 除了“任务类型”和“云主机名称”，其他参数都可以修改。
    注意：
    “保留份数”，只允许调大，不允许调小。以防止调小保留份数时，无法保证整体的快照数量。
  3. 修改完之后，单击“确定”按钮使参数生效。

新建定时备份任务

前提条件

  • 云主机备份存储设备信息已配置完成，存储设备剩余容量必须满足备份需求。
    备份存储设备在“系统管理>运营管理 >存储管理”中进行配置。
  • 云主机挂载的云硬盘状态为“可用”或“使用中”时才能进行备份创建。
    云硬盘状态在“虚拟资源管理>计算 >存储”中查看。

操作步骤

  1. 在云管平台的Web控制台上，选择“产品和服务 >系统管理 >运维管理”，进入运维管理导航菜单。
  2. 在左侧导航菜单中单击“计划任务”，进入计划任务页面。
  3. 单击“新建任务”按钮。
  4. 设置定时备份任务的各项参数。
  5. 单击“确定”按钮，在计划任务列表区可以查看任务信息。

修改定时备份任务

  1. 在计划任务列表页，勾选要编辑的定时备份任务，单击“编辑任务”按钮。
  2. 除了“任务类型”和“云主机名称”，其他参数都可以修改。
    注意：
    “保留份数”，只允许调大，不允许调小。以防止调小保留份数时，无法保证整体的备份数量。
  3. 修改完之后，单击“确定”按钮使参数生效。
