云主机组

当期望创建的云主机与某些特定的云主机分布在同一个物理节点上，以便这些云主机之间的通讯更快速方便时，可以把它们都放进同一个组；当期望创建的云主机与某些特定的云主机分布在不同的物理节点上，以便这些云主机之间实现一定的隔离，并且平衡各物理节点的资源使用率时，也可以把它们都放进同一个组。这些场景下所说的同一个组，即云主机组。

  • 亲和性：云主机组里的所有云主机分布在同一个物理节点上。
  • 反亲和性：云主机组里的所有云主机分布在不同的物理节点上。

新建云主机组

  1. 在云管平台的Web控制台上，选择“产品和服务 >虚拟资源管理 >计算”，进入计算导航菜单。
  2. 在左侧导航菜单中单击“云主机组”，进入云主机组页面。
  3. 单击“新建”按钮，填写云主机组名称和描述，选择亲和性规则。
  4. 单击“确定”按钮。云主机组创建完成后，在创建云主机时，可根据需求选择云主机组。

查看云主机组详情

在云主机组列表的“名称”列，单击待查看的云主机组，即可跳转到云主机组详情页面。查看该云主机组成员信息。

删除云主机组

在云主机组列表区，选择待删除的云主机组，单击“删除”按钮，在弹出的确认框中单击“确定”按钮，即可删除云主机组。

说明：
删除云主机组，对组内云主机无影响。
