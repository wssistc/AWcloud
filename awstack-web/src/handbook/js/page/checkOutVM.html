查看云主机

云主机列表

在云主机列表区，可以看到所有属于本账号的云主机。

可以根据需要筛选数据中心、部门及项目，查看云主机列表信息。

云主机详细信息

在云主机列表的“云主机名称”列，单击待查看的云主机，即可跳转到云主机详情页面。
云主机详细信息支持查看云主机的配置信息、监控详情、日志信息。

  • 云主机详情

基础信息包括云主机名称、镜像、配置（CPU、内存、硬盘）、云主机运行状态、IP（内网IP、公网IP）、安全组、密钥、云主机的创建时间、云硬盘及到期时间。

  • 云主机监控信息

监控信息提供云主机的监控图像，包括CPU、内存、磁盘、网卡。支持按时间筛选并导出文件。

  • 云主机日志信息

在“日志信息”页签，可查看云主机操作系统的启动日志。
