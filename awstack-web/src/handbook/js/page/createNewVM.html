新建云主机

支持“自定义配置”、“快捷配置”和“ISO安装”三种模式创建云主机，根据实际需要选择。以下分别进行描述。

自定义配置

前提条件

新建云主机前，需完成部门、项目、用户、安全组和网络信息配置。

操作步骤

  1. 在云管平台的Web控制台上，选择“产品和服务 > 虚拟资源管理 > 计算”，进入计算导航菜单。
  2. 在左侧导航菜单中单击“云主机”，进入云主机页面。
  3. 单击“新建”按钮，配置云主机基本信息。
  4. 单击“下一步”，配置云主机选择源存储。
  5. 单击“下一步”，配置网络信息。
  6. 确认配置信息，单击“新建云主机”按钮，完成创建。

云主机创建完毕后可以在云主机列表查看。云主机会自动连接到私有网络、路由器上，分配到IP地址，并享有安全组服务。

后续操作

当云主机创建失败时，需要手动删除关联的云硬盘。如不删除，则云硬盘将会占用配额并持续计费，造成用户损失。

快捷配置

前提条件

新建云主机前，需完成部门、项目、用户、安全组和网络信息配置。

操作步骤

  1. 在云管平台的Web控制台上，选择“产品和服务 >虚拟资源管理 >计算”，进入计算导航菜单。
  2. 在左侧导航菜单中单击“云主机”，进入云主机页面。
  3. 单击“新建”按钮，选择“快捷配置”页面，配置参数。
  4. 完成页面参数配置后，单击“创建”按钮。

云主机创建完毕后可以在云主机列表查看。云主机会自动连接到私有网络、路由器上，分配到IP地址，并享有安全组服务。

后续操作

当云主机创建失败时，需要手动删除关联的云硬盘。如不删除，则云硬盘将会占用配额并持续计费，造成用户损失。

ISO安装

前提条件

用于创建云主机的ISO镜像必须是可以进行安装操作系统的镜像。且已通过“上传镜像”将ISO文件上传云管平台。

操作步骤

  1. 在云管平台的Web控制台上，选择“产品和服务 >虚拟资源管理 >计算”，进入计算导航菜单。
  2. 在左侧导航菜单中单击“云主机”，进入云主机页面。
  3. 单击“新建”按钮，选择“ISO安装”页面，配置参数。
  4. 当云主机创建失败时，需要手动删除关联的云硬盘。如不删除，则云硬盘将会占用配额并持续计费，造成用户损失。
  5. 完成页面参数配置后，单击“确认新建”按钮。

后续操作

云主机创建完毕后可以在云主机列表查看。用户需要通过控制台安装ISO镜像中的操作系统，并设置网络IP等操作。
