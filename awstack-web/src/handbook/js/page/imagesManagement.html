镜像库管理

云管平台支持多次迁移镜像库到存储设备。后续创建的镜像文件会直接保存至共享存储，操作可以进行多次，可以将本地存储迁到共享存储，但不能从共享存储迁回本地。

注意：
镜像迁移过程中不能进行与镜像相关的操作，如通过镜像新建云主机、删除镜像等。镜像库迁移失败后，可以重新迁移，但此时不能删除异构存储。只有迁移成功后，才可以删除未使用的异构存储。

  1. 在云管平台的Web控制台上，选择“系统管理 > 运维管理”，进入运维管理导航菜单。
  2. 在左侧导航菜单中单击“镜像库管理”，进入镜像库管理页面。
  3. 在下拉列表中选择数据中心，并选择镜像库迁移的目的地址。
  4. 单击“开始迁移”按钮。
