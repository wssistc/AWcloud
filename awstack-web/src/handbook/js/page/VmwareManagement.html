VMware管理

云管平台还提供了VMware纳管。企业可以通过一套云管平台同时管理多种云资源，帮助企业既能实现云的灵活，又能保留VMware的企业级功能。
说明：
本云管平台只对接VMware vsphere 5.5u3版本。

VMware系统设置

登录VMware之前，需要在云管平台上设置VMware参数，否则无法看到VMware入口。参数设置如下：

  1. 在云管平台的Web控制台上，选择“产品和服务 >系统管理 >系统设置”，进入系统设置导航菜单。
  2. 在左侧导航菜单中单击“混合云设置”，进入混合云设置页面。
  3. 下拉选择云类型为“VMWARE”，并输入对应云类型的API密钥。
  4. 单击“确定”按钮。
  5. VMware参数设置成功后，云管平台的“产品和服务”页面增加“VMware”导航菜单。
  6. 单击“VMware”页签，页面跳转到VMware纳管界面。云管平台支持对VMware的数据中心、集群、主机和虚拟机进行纳管。
