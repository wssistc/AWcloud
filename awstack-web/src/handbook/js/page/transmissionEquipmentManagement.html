透传设备管理

加载透传设备

前提条件

  • 物理机上已挂载透传设备。
  • 云主机处于运行/关机的状态下进行操作。

操作步骤

  1. 在云管平台的Web控制台上，选择“产品和服务>系统管理 >运维管理”，进入运维管理导航菜单。
  2. 在左侧导航菜单中单击“透传设备管理”，进入透传设备管理页面。
  3. 单击“加载”按钮，进入加载信息页面。
  4. 选择云主机名称、设备类型、待加载的设备和USB设备兼容性，单击“确定”按钮。
    设备被加载到云主机后，会一直属于该云主机。

卸载透传设备

  1. 在透传设备列表选择需要卸载设备的云主机，单击“卸载”按钮。
  2. 在弹出的确认框中单击“确定”按钮，卸载该设备。
    卸载之后，设备可以重新被加载。且卸载之前不能拔出设备。
