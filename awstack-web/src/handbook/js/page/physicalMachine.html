物理机

申请物理机

前提条件

  • 功能管理中物理机管理功能已安装成功。详细信息请参加“11.3.5.2添加物理机管理”。
  • 纳管资源池物理机中有符合需求的物理机且完成初始化操作。
  • 申请物理机前已完成镜像和网络信息配置。

操作步骤

  1. 在云管平台的Web控制台上，选择“产品和服务 >物理资源管理 >物理机”，进入物理机页面。
  2. 单击“申请”按钮，填写物理机信息。
  3. 单击“确定”按钮。

启动物理机

当物理机处于关机状态的时候，可以启动物理机。
物理机的启动支持批量操作。

说明:
只有处于关机状态的物理机，启动按钮才可以使用。
如果进行批量操作，被选中的物理机中，若有不是关机状态的，启动按钮依然不可用。

在物理机列表中，勾选待操作的物理机，单击“启动”按钮，然后在弹出的确认框单击“确定”按钮即可。

登录控制台

物理机申请成功后，在物理机列表，选择需要进行连接操作的物理机，单击“控制台”按钮。在控制台输入物理机的登录账号，通过IPMI对物理机进行连接操作。

说明:
只有处于“运行”并“未锁定”状态的物理机，控制台按钮才可以使用。仅admin可以登录物理机的控制台。登录控制台时浏览器建议使用Internet Explorer 14+、Mozilla Firefox 43+、Google Chrome 47+、Safari 9+。

系统重装

在物理机列表中，勾选待操作的物理机，单击“更多操作”按钮并在下拉列表中选择“系统重装”，然后在弹出的确认框单击“确定”按钮即可。

注意：
系统重装会导致数据丢失，重装系统前请备份好相关数据。

关闭物理机

当物理机处于运行状态的时候，可以关闭物理机，节省费用。
物理机的关闭支持批量操作。

说明:
只有处于运行状态的物理机，关闭按钮才可以使用。
如果进行批量操作，被选中的物理机中，若有不是运行状态的，关闭按钮依然不可用。

在物理机列表中，勾选待操作的物理机，单击“更多操作”按钮并在下拉列表中选择“关机”，然后在弹出的确认框单击“确定”按钮即可。

重启物理机

当物理机处于运行状态的时候，可以对物理机进行重启操作，满足用户的需求。物理机的重启支持批量操作。

说明:
只有处于运行状态的物理机，才能够进行重启操作。

在物理机列表，勾选待操作的物理机，单击“更多操作”按钮并在下拉列表中选择“重启”，在弹出的确认框单击“确定”按钮即可。
