数据中心管理

查看数据中心列表

在云管平台的Web控制台上，选择“产品和服务 >系统管理 > 运维管理”，进入运维管理导航菜单。在左侧导航菜单中单击“数据中心管理”，进入数据中心管理页面。在数据中心管理页面，列表区显示当前系统内的数据中心的信息。

  • 注册码：部署数据中心时，灌装物理服务器后注册到云管平台，获取注册码。
  • 虚拟化节点数量：可供部署云主机的宿主机节点数。
  • 容器节点数量：可供部署容器的宿主机节点数。

新增数据中心

说明：
使用下沉模式部署云管平台时，不允许进行添加数据中心的操作。

  1. 在数据中心管理页面，单击“新增”按钮。
  2. 填写数据中心名称，单击“确定”按钮。
  3. 选择“立即部署”或“稍后部署”。
    选择“稍后部署”，在数据中心列表区单击数据中心名称时，弹出提示框，需要是否立即部署。
  4. 进入数据中心部署页面。依次完成主机配置、网络配置、账号设置和信息汇总。在账号设置页面不用配置用户信息。
  5. 配置完成后单击“开始部署”。
  6. 数据中心安装部署完成后重新登录云管平台，进行各类操作时可以切换不同数据中心。

删除数据中心

删除数据中心前应确认部署在该数据中心上的所有云主机均已全部清除。

在列表区勾选待删除的数据中心，单击“删除”按钮，在弹出的确认框里单击“确定”按钮。删除后，所有数据将被清空。

关闭数据中心

关闭数据中心前应确认部署在该数据中心上的所有云主机均已安全关机，否则在关闭数据中心时会导致云主机数据丢失。

在列表区勾选待关闭的数据中心，单击“一键关闭”按钮，在弹出的确认框里单击“确认并关闭环境”按钮。将关闭该数据中心云管平台所管理的所有物理宿主机。
