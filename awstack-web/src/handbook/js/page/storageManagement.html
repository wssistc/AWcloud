存储管理

添加备份存储

说明：
云主机只能添加一个备份存储。

前提条件

已获取对接的外部存储设备信息。

操作步骤

  1. 在云管平台的Web控制台上，选择“产品和服务>系统管理 >运维管理”，进入运维管理导航菜单。
  2. 在左侧导航菜单中单击“存储管理”，进入存储管理页面。
  3. 单击“添加”按钮。
  4. 填写存储名称和描述信息，在存储用途的下拉列表中选择“云主机备份资源池”，在存储设备下拉列表中选择“NFS”或“Ceph”。
  5. 单击“确定”按钮。

添加后端存储

前提条件

已获取对接的外部存储设备信息，用户名、密码、IP地址等。

操作步骤

  1. 在云管平台的Web控制台上，选择“产品和服务 >系统管理 >运维管理”，进入运维管理导航菜单。
  2. 在左侧导航菜单中单击“存储管理”，进入存储管理页面。
  3. 单击“添加”按钮。
  4. 在存储用途的下拉列表中选择“云主机存储资源池”，配置云主机后端存储设备信息。

         注意：

  • 两节点软件交付模式下，不允许对接超融合Ceph。
  • 由于当前云管平台不对外部Ceph存储进行使用状态监控，当外部Ceph使用率过大时可能会影响云管平台功能的使用。
  • 当存储协议为“FC”时，则不需要配置主机。
  • 连接通用iscsi时，当有多个磁盘可供选择时，所有节点必须选择相同大小的磁盘才能对接成功。
  1. 配置主机信息，包括存储网信息和各节点网卡信息。
  2. 在主机列表区勾选节点，单击“磁盘配置”按钮，选择超融合Ceph的磁盘。完成所有节点磁盘配置。
  3. 单击“网卡配置”按钮，查看网卡配置信息。
  4. 单击“下发网络配置”按钮，使配置生效。
  5. 单击“验证存储连通性”按钮，验证各节点网络联通性。
  6. 单击“保存”按钮。

节点存储管理

当云主机存储资源池对接的设备类型为“通用FC”或“通用ISCSI”时，数据中心新增节点纳入集群后，需要进行节点存储设备对接。

当对接的存储设备类型为“通用FC”时，单击名称列的存储名称，进入存储详情页面，选择“节点存储管理”页签。在节点列表区，选择新增的节点，单击“磁盘管理”按钮，选择挂载FC存储磁盘。

当对接的存储设备类型为“通用ISCSI”时，单击名称列的存储名称，进入存储详情页面，选择“节点存储管理”页签。单击“发现目标（target）”，查看节点信息，然后单击“登陆并确认ISCSI存储设备”按钮，当新增节点有多个磁盘可供选择时，必须选择其他节点相同大小的磁盘才能对接成功。

修改存储

注意：
使用中存储设备不允许修改。

在存储管理页面，勾选待修改的存储，单击“编辑”按钮，修改存储名称和描述信息。

删除存储

注意：
删除存储前需确认该存储未被使用。

在存储管理页面，勾选待删除的存储，单击“删除”按钮，在弹出的确认框里单击“确定”按钮。
