IPSec VPN

新建VPN

前提条件

本地路由器和目标路由器均已关联子网。

操作步骤

  1. 在云管平台的Web控制台上，选择“产品和服务 >虚拟资源管理 >网络”，进入网络导航菜单。
  2. 在左侧导航菜单中单击“IPSec VPN”，进入IPSec VPN页面。
  3. 单击“新建”按钮，设置基础信息。
  4. 单击“下一步”按钮，配置站点连接信息。
  5. 单击“新建”按钮，完成VPN1创建。新创建的VPN1状态为“关闭”。
  6. 参考之前的步骤，继续创建另一个VPN2。

注意：

配置站点连接信息时，将VPN1的本地网络信息配置为VPN2的目标网络信息，将VPN1的目标网络信息配置为VPN2的本地网络信息。

IPSec VPN功能会与公网IP功能冲突，建立站点连接之后请检查站点双方私有子网内云主机是否绑定公网IP，如果有则需要解除公网绑定。

创建完成后VPN2状态为“创建中”，大约5秒左右为“运行”，刷新页面，VPN1状态变为“运行”。此时，本地网络和目标网络可以进行通讯。

修改VPN

在VPN列表，勾选待操作的VPN，单击“编辑”按钮，修改完成后单击“确定”按钮。

注意：
如果修改后的VPN状态为“关闭”，则可能由于修改了目标网关地址导致VPN可不用。
