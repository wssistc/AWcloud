身份管理

身份管理功能按照企业日常管理的视角，支持创建符合本企业组织结构的部门；并支持基于部门进行资源分配；支持基于部门、项目以及项目成员进行角色（包括部门管理员、项目管理员、普通用户）划分；支持基于角色进行云管平台系统功能的授权。

企业可以实现按需创建不同权限、不同配额的子账号，并由各子账号独立管理本权限内的所有云资源。满足了企业多级权限管理的需求，为企业不同部门不同项目管理其资源提供了便利。

  • 超级管理员：默认为admin用户，为系统自动生成且不能修改，具有整个平台的最高权限。
  • 部门管理员：权限仅次于超级管理员，负责创建项目，管理部门成员。
  • 项目管理员：权限仅次于部门管理员，主要负责管理项目资源配额。
  • 普通用户：为最基层组织成员，为云计算资源的真正使用者，不具备管理账户的能力。

新建部门

  1. 在云管平台的Web控制台上，选择“产品和服务 >身份管理 >部门管理”，进入部门管理页面。
  2. 单击“新建部门”按钮。
  3. 设置部门的名称、描述，并给部门分配配额。
  4. 完成以上设置后，单击“确定”按钮。操作成功后，可以在列表区查看。

新建项目

前提条件

如果不使用云管平台提供的“默认部门”，则需要新建部门。

操作步骤

  1. 在云管平台的Web控制台上，选择“产品和服务 >身份管理 >项目管理”，进入项目管理页面。
  2. 在页面右侧单击“新建项目”按钮。
  3. 设置项目的名称、描述，并给项目分配配额。
    项目配额不能大于部门配额。
  4. 完成以上设置后，单击“确定”按钮。操作成功后，可以在列表区查看。

给项目分配用户

前提条件

该部门下存在用户。如果没有用户，请在“身份管理 >用户管理”中新建用户。

操作步骤

  1. 在项目列表区，勾选待操作的项目，单击“分配用户”按钮。
  2. 在左侧的“未分配的用户”中选择待分配的用户，将用户添加到右侧的“已分配的用户”。
  3. 在“已分配的用户”中，单击右侧蓝色字体区域，在下拉列表中勾选用户角色。
  4. 单击“确定”按钮。

删除项目

注意：
如果项目下面有资源，此项目不能被删除。资源包括：云主机、快照、云硬盘、备份、公网IP、路由器、网络、子网、安全组、回收站中的云主机。

  1. 在项目列表区，勾选待操作的项目，然后单击“删除”按钮。
  2. 在弹出的确认对话框，单击“确定”按钮。

新建用户

前提条件

  • 新建部门
  • 新建项目
    如果需要创建项目管理员或者普通用户，需要提前新建项目。

操作步骤

  1. 在云管平台的Web控制台上，选择“产品和服务 >身份管理 >用户管理”，进入用户管理页面。
  2. 单击“新建用户”按钮。
  3. 选择角色，然后单击“下一步”按钮。
  4. 选择部门、项目，然后单击“下一步”按钮。
  5. 设置基本信息，包括登录名称、密码、手机和Email。
  6. 完成以上设置后，单击“新建”按钮。操作成功后，新建的用户就会展示在用户列表中。

锁定或激活用户

管理员可以锁定或者激活普通用户。在用户列表区单击待锁定或激活的用户所对应的“锁定”或“激活”按钮。

注意：
对用户执行锁定操作后，该账号将无法使用。

删除用户

注意：
不允许删除admin。

用户被删除后，该用户之前创建的资源不受影响，还可以继续使用。

  1. 在用户列表区，选择需要删除的用户，单击“删除”按钮。
  2. 在弹出的对话框单击“确定”按钮。

角色管理

云管平台为不同角色设置了不同的管理权限，角色管理提供了权限调整功能。需要注意，权限调整只能在现有基础上减少角色的管理范围。而且，超级管理员不支持此功能。

  1. 在云管平台的Web控制台上，选择“产品和服务 >身份管理 >角色管理”，进入角色管理页面。
  2. 勾选待调整的角色，单击“编辑”按钮。
  3. 去勾选需要屏蔽的功能模块，然后单击“确定”按钮。操作成功后，重新登录云管平台，该角色将无法看到此功能。
