用快照创建云硬盘

基于快照创建新的云硬盘，这样云硬盘在初始状态就具有快照中的数据。

数据盘快照和系统盘快照均支持此功能。

  1. 在快照列表页，勾选用来创建云硬盘的快照，单击“创建云硬盘”按钮。
  2. 在弹出的页面中填入“云硬盘名称”、“描述信息”、“配置”，并根据实际需要勾选“共享卷”。
  3. 单击“确定”按钮。操作成功后，可以在云硬盘的列表区查看。

通过系统盘快照新建云主机

只有系统盘快照支持创建云主机。

  1. 在系统盘快照列表页，勾选用来创建云主机的快照，单击“新建云主机”按钮。
  2. 设置云主机各项参数，云主机参数的设置方法同前面的新建云主机。
  3. 设置完成后，单击“确定”按钮。
