软件授权

查看授权信息

  1. 在云管平台的Web控制台上，选择“产品和服务 >系统管理 >系统设置”，进入系统设置导航菜单。
  2. 在左侧导航菜单中单击“软件授权”，进入软件授权页面。
  3. 可以在界面上查看激活有效期限。云管平台一般提供一个月免费试用期，如果需要延长有效期，需要购买使用权以获取有效License。

获取机器码

  1. 在界面上单击“获取机器码”。
  2. 在获取机器码的对话框中，单击右边的“复制”快捷键，将本环境授权参数快速复制。
  3. 按照以下邮件格式发送本集群环境相关信息至license@awcloud.com，可收到一封带有激活文件的回复邮件，激活文件名通常为license.pem。

导入激活文件

前提条件

已收到带有激活文件的邮件，激活文件名通常为license.pem。云管平台支持按照功能模块制作license，将功能列表保存在license文件中。

操作步骤

  1. 在界面上单击“导入激活文件”。
  2. 单击“选择文件”，选择本地保存的激活文件，然后单击“确定”按钮。
  3. 导入成功后，会查看到当前的授权信息。
    激活有效期是指平台可用的期限，过期后将不能再创建云主机。
    服务有效期是指对平台进行技术支持的期限，过期后将失去技术支持，但不影响正常使用。
  4. 单击“SN号”下的详情，可以查看到当前平台的SN号，寻求技术支持时需要提供。
  5. 重新登录系统后，功能菜单将重新显示License文件定义的功能列表。
