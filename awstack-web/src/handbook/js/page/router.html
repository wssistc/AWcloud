路由器

新建路由器

前提条件

  • 私有网络已创建完成。
  • 项目和部门剩余配额满足需求。

操作步骤

  1. 在云管平台的Web控制台上，选择“产品和服务 >虚拟资源管理 >网络”，进入网络导航菜单。
  2. 在左侧导航菜单中单击“路由器”，进入路由器页面。
  3. 在路由器页面的右上方，单击“新建”按钮。
  4. 配置路由器信息。
  5. 单击“确定”按钮。操作成功后，可以在列表区查看。

设置/删除网关

网关（Gateway）就是一个网络连接到另一个网络的“关口”，实质上是一个网络通向其他网络的IP地址。这里的网关就是与路由器相连的外部网络的IP地址。

注意：
只有安装部署阶段创建了Neutron的外部网络，才可以执行这里的设置网关操作。
路由器关联的子网下的云主机没有绑定公网IP时，才可以执行删除网关操作。删除网关后，云主机将无法访问外网。

  1. 在路由器列表页，勾选待操作的路由器，单击“设置网关”按钮，从下拉列表中选择安装部署阶段创建的外部网络，云管平台会自动设置此网络的IP地址为路由器的网关，然后单击“确定”按钮。
  2. 在路由器列表页，勾选待操作的路由器，单击“删除网关”按钮，在弹出的页面中单击“确定”按钮即可。

新建端口转发

当用公网IP地址有限的时候，可通过虚拟路由器的端口转发功能，通过路由器的网关接口地址和端口号访问某个云主机开放的端口号。

注意：
当云主机已绑定公网IP时，端口转发功能会失效。

  1. 在路由器列表勾选待添加静态路由的路由器，单击“端口转发”按钮并在下拉列表中选择“新建”。
  2. 配置端口转发信息，将云主机私网IP地址的一个端口映射到公网IP的一个端口上。
  3. 单击“确定”按钮。

删除端口转发规则

  1. 在路由器列表勾选待添加静态路由的路由器，单击“端口转发”按钮并在下拉列表中选择“删除”。
  2. 在弹出的提示框里单击“确认”按钮。

注意：
删除云主机时，针对该云主机设置的端口转发规则将会同步删除。

删除路由器

注意：
删除路由器之前，需要先解除路由器绑定的端口，可以通过前面的“解除关联子网”来操作。加载了防火墙的路由器不允许被删除。

在路由器列表页，勾选要删除的路由器，单击“删除”按钮，在弹出的确认框中单击“确定”按钮。
