配额管理

部门配额

  1. 在云管平台的Web控制台上，选择“产品和服务>系统管理 >运营管理”，进入运营管理导航菜单。
  2. 在左侧导航菜单中单击“配额管理”，进入配额管理页面。
  3. 单击“部门配额”按钮，在输入框依次输入配额数量。“是否可配置”表示新建部门时，部门管理员能否调整配额，勾选表示可以调整；不勾选表示不能调整，保持默认配置。
  4. 单击“确定”按钮。

说明：
可以通过拖拽左侧上下图标的方式对资源名称进行排序。

项目配额

  1. 在云管平台的Web控制台上，选择“产品和服务 >系统管理 >运营管理”，进入运营管理导航菜单。
  2. 在左侧导航菜单中单击“配额管理”，进入配额管理页面。
  3. 单击“项目配额”按钮，在输入框依次输入配额数量。“是否可配置”表示新建项目时，项目管理员能否调整配额，勾选表示可以调整；不勾选表示不能调整，保持默认配置。
  4. 单击“确定”按钮。
