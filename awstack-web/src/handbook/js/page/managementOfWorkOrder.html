工单管理

工单申请

前提条件

供用户进行工单申请的类型必须由admin用户进行流程绑定。具体操作请参见“11.2.6.1.1新建工作流”。

操作步骤

  1. 在云管平台的Web控制台上，选择“产品和服务 >工单管理 >工单申请”，进入工单申请页面。

admin仅支持信息咨询类工单。

供其他非admin角色的用户进行工单申请的类型如下：

2. 选择工单类型，单击“由此申请”，填写各类申请的信息。

  • 信息咨询

            通过下拉列表选择收件人，并输入要咨询的信息（包括标题和描述），单击“确定”按钮。操作成功后，收件人可以在“我的工单 >待处理”中查看到此工单。

  • 资源申请

            输入申请资源相关信息，选择部门名称和项目名称，按照实际需求输入云主机、路由、CPU核、公网IP、磁盘快照和内存等，均为必填项，单击“确定”按钮。操作成功后，审批人可以在“我的工单 >待处理”中查看到此工单。

待处理工单

在待处理工单页面对需要由当前登录用户审批的工单进行操作。

前提条件

接收审批结果的通知邮件需要发件人的邮件服务器配置正确且申请人的邮箱地址填写正确。

操作步骤

  1. 在云管平台的Web控制台上，选择“产品和服务 >工单管理 >待处理工单”，进入待处理工单页面。
  2. 选中要处理的工单，在最后一栏操作列，单击“审批”或“回复”按钮。
  3. 对于咨询类工单填写回复内容后单击“回复”按钮即可。对于资源申请类工单，根据实际情况填写审批结果。
  4. 审批完成后，申请人会收到审批结果通知邮件。

我的申请

  1. 在云管平台的Web控制台上，选择“产品和服务 >工单管理 >我的申请”，进入我的申请页面。
  2. 查看当前用户所申请的工单信息及状态。当已申请工单比较多时，可以在搜索过滤输入框，输入查询的关键字，即可看到所有符合此关键字的工单。
  3. 对于资源申请类工单，可以单击“撤销”按钮，撤销选中的工单。
  4. 对于信息咨询类工单，可以单击“关闭”按钮，关闭选中的工单。

我的审批

在云管平台的Web控制台上，选择“产品和服务 >工单管理 >我的审批”，进入审批页面。包含“我未处理的工单”和“我已处理的工单”。

在工单列表区，单击工单编号或审批按钮，查看工单详细信息及审批进度。

对于待审批的工单，咨询类工单填写回复内容后单击“回复”按钮即可。资源申请类工单，根据实际情况填写审批结果。
