安全设置

访问策略

通过设置访问的黑白名单，过滤云管平台访问权限，保障云管平台的安全。

新建策略

  1. 在云管平台的Web控制台上，选择“产品和服务 >系统管理 >系统设置”，进入系统设置导航菜单。
  2. 在左侧导航菜单中单击“安全设置”，进入安全设置页面。
  3. 选择“访问策略页签”。
  4. 单击“新建”按钮，在弹出的对话框里设置访问策略相关信息。
  5. 单击“确定”按钮。操作成功后，可以在访问控制策略列表区查看。

登录失败策略

  1. 在云管平台的Web控制台上，选择“产品和服务 >系统管理 >系统设置”，进入系统设置导航菜单。
  2. 在左侧导航菜单中单击“安全设置”，进入安全设置页面。
  3. 选择“登录失败策略”页签，打开“登录失败锁定策略”开关。
  4. 单击“设置”按钮，配置策略信息。
  5. 设置“连续登录失败次数”和“锁定用户登录时间”，单击“确定”按钮。
  6. 在用户列表区选择用户，单击“删除”按钮，用户将不受登录锁定策略限制。
  7. 如需关闭失败锁定策略，则关闭“安全设置”按钮。
