弹性扩展

创建云主机

弹性扩展集群的云主机扩展是通过监控资源的使用率，来实现自动扩展的。如果资源使用率未达到阈值，就无法扩展集群。但是很多业务增长是非常快速的，自动扩展可能响应不及时。云管平台支持手动添加云主机，实现集群的扩展。

  1. 在云管平台的Web控制台上，选择“产品和服务>虚拟资源管理 >弹性扩展”，进入弹性扩展导航菜单。
  2. 在左侧导航菜单中单击“集群管理”，进入弹性集群管理页面。
  3. 在集群列表区，勾选待添加云主机的集群，  单击“添加云主机”按钮，配置云主机信息。
  4. 单击“确定”按钮。

删除弹性扩展策略

注意：
如该弹性扩展策略已被集群使用，则不可删除。

在扩展策略列表区，勾选需要删除的策略，单击“删除”按钮，在弹出的确认框里单击“确定”按钮。
